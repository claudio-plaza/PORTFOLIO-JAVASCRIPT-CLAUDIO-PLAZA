# Portfolio JavaScript - Claudio Plaza

Este es un portfolio con varios ejercicios en JavaScript, pensados para practicar desde conceptos básicos hasta cosas con APIs y almacenamiento en el navegador.

Es un README sencillo: ni muy técnico ni muy formal — lo justo para que quien lo vea entienda qué hay aquí.

Proyectos incluidos (resumen):

- Ejercicio 1: Calculadora básica — operaciones entre dos números.
- Ejercicio 2: Adivina el número — juego simple con varios intentos.
- Ejercicio 3: Tablas de multiplicar — genera tablas según el número ingresado.
- Ejercicio 4: Conversor de temperaturas — entre °C, °F y K.
- Ejercicio 5: Lista de compras — muestra productos y calcula total.
- Ejercicio 6: Agenda de contactos — lista básica de contactos.
- Ejercicio 7: Reloj digital — muestra la hora y se actualiza cada segundo.
- Ejercicio 8: Calculadora gráfica — interfaz con botones para operaciones.
- Ejercicio 9: Lista de tareas — añadir y eliminar tareas en la UI.
- Ejercicio 10: Formulario de registro — validaciones simples (nombre, email, contraseña).
- Ejercicio 11: Cronómetro — iniciar, pausar y reiniciar.
- Ejercicio 12: Buscador de clima — usa OpenWeatherMap para mostrar datos de una ciudad.
- Ejercicio 13: Buscador GitHub — muestra información pública de un usuario de GitHub.
- Ejercicio 14: Notas — guardar notas en `localStorage` del navegador.
- Ejercicio 15: Gráficas — gráficos interactivos con Chart.js (añadir datos y ver estadísticas).

Cómo probarlo localmente (sencillo):

1. Abre una terminal en la carpeta del proyecto.
2. Arranca un servidor simple (recomendado para evitar problemas con `file://`):

```powershell
cd "c:\Users\Gaton\Documents\web\casa del futuro\PORTFOLIO-JAVASCRIPT-CLAUDIO-PLAZA"
python -m http.server 8000
```

3. Abre en el navegador: `http://localhost:8000/` y prueba los ejercicios.

Notas y cosas a mejorar (opcional):

- Algunos ejercicios usan APIs externas — puede que necesites revisar las keys o el estado del servicio.
- Hay archivos en `scripts/` numerados por ejercicio. Puedes separar cada ejercicio en su propia página en `html/` si quieres.
- Este proyecto está montado para aprender y mostrar ejemplos; si quieres que lo deje más profesional o que haga deploy en GitHub Pages, dímelo y lo hago.
